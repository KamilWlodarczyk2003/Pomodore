
# Pomodoro Timer

Pomodoro timer that can optimize you study efficiency.  

Step 1: Pick a task  
Step 2: Run this program  

Program technique:  
* 25 minutes timer where you work/study
* 5 minutes break
* Every 4 cycles it sets 25 minutes for a long break
